# Install

Install in a virtual environment:

```
pip install .
```

# Run

To run:

```
ubitflashtool -s extracted_script.py
```

To run the GUI:

```
ubitflashtool --gui
```

or from this directory:

```
python -m ubitflashtool --gui
```
